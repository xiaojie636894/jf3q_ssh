
一、导包说明：
导入此文件夹的所有jar包到classpath中。


二、jar包来源：
此文件夹中的所有jar包全部来自struts-2.3.16.3-all.zip，解压后在lib文件夹中都可以找到。
注意：但并不是lib文件夹中的全部jar都需要复制过来，只根据需要复制相应的jar过来即可。
